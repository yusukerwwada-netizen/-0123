## 🌟 アプリの主な機能

このアプリは「ただクイズを解く」だけでなく、クラウドデータベースを活用した以下の機能を備えています。

1. **クラウドからの出題**: 
   Supabase上のテーブルから問題をリアルタイムに取得します。問題の追加や修正もデータベース側で即座に反映されます。
2. **5問完結チャレンジ**: 
   多数ある問題の中から自動的に5問に絞って出題され、テンポよく学習を進められます。
3. **詳細なフィードバック**: 
   回答後すぐに正誤判定を行い、正解の提示とともに「なぜそうなるのか」という詳しい解説を表示します。
4. **学習履歴の永続保存**: 
   従来の簡易アプリと異なり、クイズ終了時のスコアと日時をクラウドに保存。アプリを閉じてもデータが消えることはありません。
5. **過去ログの可視化**: 
   結果画面にて、自分（または他のユーザー）が過去に挑戦した最新5件の履歴を確認できます。



## 🛠 使用技術
- **Frontend**: Streamlit
- **Database**: Supabase (PostgreSQL)
- **Language**: Python

## 📋 データベース構造
本アプリはSupabase内に以下の2つのテーブルを利用しています：
- `quiz_questions`: クイズの問題、選択肢、正解、解説を格納。
- `quiz_logs`: ユーザーの挑戦結果（スコア、実施日時）を格納。

## 🚀 セットアップ（開発者向け）
1. `pip install streamlit supabase` でライブラリをインストール。
2. `.streamlit/secrets.toml` にSupabaseのURLとAPI Keyを設定。
3. `streamlit run app.py` で起動。
